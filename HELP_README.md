## 🎯 Exercise Overview

This is a 5-step interactive exercise where you'll learn to use GitHub Copilot by working on a real web application:

### Step 1: Hello Copilot 👋
- Create a Codespace
- Get familiar with Copilot modes
- Create the `accelerate-with-copilot` branch

### Step 2: Getting Work Done 🔧
- Fix a duplicate signup bug
- Generate sample data with Copilot
- Commit changes with AI-generated messages

### Step 3: Agent Mode 🤖
- Use Agent Mode for multi-step tasks
- Add participant lists to UI
- Implement unregister functionality

### Step 4: Plan Agent 🧭
- Design a testing strategy
- Generate comprehensive test suite
- Ensure all tests pass

### Step 5: Pull Request Features 🚀
- Create a PR with Copilot summary
- Request Copilot code review
- Merge your changes

## 🚀 Getting Started

### Quick Start (Recommended Path)

1. **Start the exercise** using the badge in the main README
2. **Follow Step 1** to create your Codespace
3. **Use QUICK_REFERENCE.md** for shortcuts and prompts as you work
4. **Refer to EXERCISE_HELP.md** if you get stuck on any step
5. **Check examples/** only if you need to see a working solution

### If You're Stuck

1. **First**: Review the relevant section in EXERCISE_HELP.md
2. **Second**: Check QUICK_REFERENCE.md for prompt patterns
3. **Third**: Look at the examples/ directory for working code
4. **Fourth**: Ask Copilot directly! Use prompts like:
   ```prompt
   I'm stuck on [specific problem]. Can you help?
   ```

## 📖 How This Exercise Works

This is an **interactive, self-paced** exercise:

- ✅ **You complete tasks in a Codespace** (not on GitHub.com)
- 🤖 **GitHub Copilot assists you** as you code
- 📝 **Automated feedback** tracks your progress
- 🎓 **Learn by doing**, not just reading

The exercise is designed to teach you how to:
- Work effectively with AI-powered coding assistance
- Use different Copilot interaction modes appropriately
- Write effective prompts to get better suggestions
- Review and validate AI-generated code
- Integrate Copilot into your development workflow

---

## 💡 Learning Tips

### Do's ✅

- **Experiment**: Try different prompts and approaches
- **Iterate**: Refine prompts if you don't get good results
- **Understand**: Read and understand AI suggestions before accepting
- **Verify**: Test code after accepting suggestions
- **Ask questions**: Use Copilot to explain code you don't understand

### Don'ts ❌

- **Don't blindly copy**: The goal is learning, not just completion
- **Don't skip steps**: Each step builds on previous learning
- **Don't give up quickly**: Iterate on prompts if needed
- **Don't forget to test**: Always verify code works as expected
